# 18. OOP

## 핵심 개념

### Rust의 객체지향적 특성
- **캡슐화**: `pub` / private 접근 제어
- **상속 대신 조합**: 트레이트로 동작 공유
- **다형성**: 트레이트 객체 (`dyn Trait`)

### 트레이트 객체
- `Box<dyn Trait>`: 런타임 디스패치 (동적)
- vtable을 통한 메서드 호출
- **객체 안전성**: `Self`를 반환하지 않는 트레이트만 가능

### 상태 패턴 예제
- 상태를 트레이트 객체로 표현
- 상태 전환을 메서드로 구현
- **Rust 방식**: 타입 시스템으로 상태를 인코딩 (컴파일 타임 검증)

## 실행

```bash
rustc oop_example.rs && ./oop_example
```
